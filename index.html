1:urllib的使用

  • urllib的下载: urllib是内置库所以不用下载

  • 初学者建议先看urllib官方文档

  • urllib.request for opening and reading URLs

  • urllib.error containing the exceptions raised by urllib.request

  • urllib.parse for parsing URLs

  • urllib.robotparser for parsing robots.txt files

  • 主要含有四个模块

  • urllib.request 打开和阅读 URLs

  • urllib.error 包含 urllib.request 抛出的异常,捕获异常

  • urllib.parse 用于处理 URL(拆分,解析,合并等)

  • urllib.robotparser 用于解析 robots.txt 文件

  • request模块的使用

  • import urllib.request
response = urllib.request.urlopen('http://www.baidu.com')    # 获取url里面的信息生成一个http response类
content = response.read().decode('utf-8')  # 通过里面的方法读取html文件
print(response.status)   # 输出状态码
print(response.getheaders())    # 获取响应头的内容(列表形式)
print(content)

  • 至于请求头的内容添加在request模块用会更加简单,在urllib就不讲了

2:requests的使用(强烈推荐这个)

  • requests属于第三方库因此需要pip命令下载(pip install requests -i https://pypi.tuna.tsinghua.edu.cn/simple)

  • import requests
# 相比于urllib这个会更加简单
response = requests.get('http://httpbin.org/get')   # 针对get请求
print(response.json())     # 获取html文件(请求信息)
print(response.cookies)  # 获取cookies信息

headers = {

  • 模仿文件上传

  • import requests
files = {'file': open('favicon.ico', 'rb')}
r = requests.post("https://httpbin.org/post", files=files)   # 上传文件
print(r.text)

  • 会话维持:如果第一个请求是用post()方法登录某个网站,第二次想获取成功登录后的自己个人的信息,你又用一次get()方法去请求个人信息页面,实际上相当于打开了俩次浏览器,是俩个完全不相关的会话,这个是不能获取到个人信息的.(可以通过设置相同的cookies来获取,但比较烦琐)

  • 举个例子(http://httpbin.org)

responese = requests.get("http://httpbin.org/cookies/set/number/123456789")  # 先用get方法登录了网站(cookies中名称是number 内容是123456789)

  • import requests
from requests.packages import urllib3

urllib3.disable_warnings()    # 用来忽略警告

  • import requests
from requests.exceptions import RequestException
import re
import time


# 先爬取一页的内容
def get_one_page(url):
    try:
        headers = {
            'User - Agent': 'Mozilla / 5.0(WindowsNT10.0;Win64;x64;rv: 85.0) Gecko / 20100101Firefox / 85.0'  # 从Windows的火狐浏览器获取的
        }
        response = requests.get(url, headers=headers)
        if response.status_code == 200:  # 判断状态码
            return response.text
        return None
    except RequestException:   # 捕捉异常
        return None
